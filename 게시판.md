#  Spring 게시판에 사용된 기술 정리

## 🚀 1. Spring Boot
- Java 기반 웹 애플리케이션 프레임워크
- 복잡한 설정 없이 빠르게 서버 실행 가능
- 내장 Tomcat으로 별도 서버 설치 필요 없음

---

## 2. Spring MVC
- Model / View / Controller 구조
- 역할을 나누어 개발하는 방식

###  구성
- Controller → 요청 처리 (HomeController)
- Model → 데이터 전달 (Post)
- View → 화면 (index.html)

---

##  3. Spring Data JPA
- 데이터베이스 작업을 쉽게 처리
- SQL 없이 CRUD 가능

###  사용 예
```java
postRepository.save(post);
postRepository.findAll();
postRepository.deleteById(id);
```

---

##  4. Hibernate
JPA 구현체 (ORM 기술)  
객체와 DB 테이블 자동 매핑  

###  사용 예
```java
@Entity
public class Post
```

##  6. Thymeleaf
HTML에서 데이터를 출력하는 템플릿 엔진  

###  사용 예
```html
<tr th:each="post : ${list}">
```

---

##  7. HTTP
웹 통신 프로토콜  

###  사용 방식
- GET → 페이지 요청  
- POST → 데이터 전송
